______________________FUNCTION IN JAVASCRIPT________________________

A function is a block of code jo specific task perform karta hai.
syntax -   
                // function definition
          function functionName ()  // () bracket for parameter
          {

            // line of code

          }
          functionName ();   // function call
Agar hum function call nahi karenge toh function nahi chalega.

function with parameter
syntax -
         function printNumber(num) {      //num yha pe parameter function ka
            console.log("printing number: ", num);
         }
        printNumber(34);                 //34 yha pe number hai jisko argument kehte hai function call ka
